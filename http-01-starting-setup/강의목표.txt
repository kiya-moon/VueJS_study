현재 애플리케이션에는 문제가 있음
데이터를 제출하면 화면상에 데이터가 표시되지만, 새로고침을 하면 데이터가 유실됨
이는 데이터를 Vue, 즉 컴퓨터의 메모리상에만 저장했기 때문
즉, 모든 컴포넌트에서 생성된 데이터가 로컬 환경 메모리에만 저장된 것
그래서 웹사이트가 새로고침할 때마다 현재 Vue 앱이 버려지고 Vue 앱의 새 인스턴스가 시작되면서 이전에 실행한 Vue 앱에 연결된 모든 데이터가 손실된다

따라서 이번 강의에서는 더미 백엔드에 데이터를 저장하는 방법을 배울 예정

node.js 등의 언어를 사용해 백엔드 서버 사이드 코드를 작성하고 데이터 베이스를 연결하는 방법도 있지만, 
이번 강의에서는 백엔드를 제공하는 서비스인 Firebase를 사용한다

Firebase를 사용하면 데이터 저장을 위해 코드를 작성할 필요가 없다.
Google에서 제공하는 서비스로 완벽한 백엔드 서비스가 사전에 작성 및 구성되어 있다
또한 무료이다

데이터베이스의 종류로는 firestore database와 realtime database가 있는데,
강의에서는 realtime database를 사용한다
